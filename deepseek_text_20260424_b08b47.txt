QUESTION 1 (Password Validator)
Write a program that:
- Takes a password as input from user
- Checks if password length is at least 8 characters
- Counts how many uppercase, lowercase, digits, and special characters (!@#$%^&*) are in the password
- Prints "Strong Password" if: length>=8 AND uppercase>=1 AND lowercase>=1 AND digits>=1
- Otherwise prints "Weak Password" with specific reason why
- Also display the password with all vowels replaced by '*'

Example Input: Hello@123
Example Output:
Length: 8 ✓
Uppercase: 1, Lowercase: 4, Digits: 3, Special: 1
Strong Password
Masked Password: H*ll*@123


QUESTION 2 (Text Formatter)
Write a program that:
- Takes a paragraph from the user
- Counts total words (split by spaces)
- Counts total sentences (split by . ! ?)
- Finds and displays the longest word in the paragraph
- Converts the entire paragraph to title case (first letter of each word capital)
- Replaces all occurrences of "bad" with "good" (case-insensitive)
- Checks if the paragraph contains the word "Python"
- Prints the paragraph in reverse order (characters reversed)

Example Input: python is fun. python is easy!
Example Output: [Your program should handle all above tasks]


QUESTION 3 (Username Generator)
Write a program that:
- Takes user's first name, last name, birth year, and favorite color
- Generates username = first 3 letters of first name + last 2 letters of last name + last 2 digits of birth year
- Converts the entire username to lowercase
- Checks if username contains any digit
- Counts how many vowels are in the username
- Reverses the username and displays it
- Creates an email = username + "@generated.com"
- Displays all results using f-strings

Example Input: First=Alexander, Last=Hamilton, Year=1995, Color=blue
Example Output: Username: aleham95, Reversed: 59mah ela, Email: aleham95@generated.com


QUESTION 4 (Message Decoder)
Write a program that:
- Takes an encoded message as input (like "H3ll0 W0rld")
- Replaces all digits with their word forms (0=zero, 1=one, 2=two, 3=three, etc.)
- Converts the entire message to uppercase
- Removes all spaces from the message
- Checks if the message is a palindrome (same forward and backward) after removing spaces
- Counts how many 'E' letters are in the final message
- Prints the decoded message and all results

Example Input: H3ll0
Example Output: Decoded: HTHREEl lZERO (show step by step)
